Converting Between Numbers and Strings

When we introduced the concatenation operator, we informed you that, if one of the operands to the operator + is a string and the other is a number, then Java converts the number to a corresponding string and then interprets + as the concatenation operator. In this case, the conversion of the number to a string is automatic. In a moment, we explain how such a conversion may be specifically ordered, but before we do, the question already arises as to how the reverse conversion may be achieved. If you have a string that names a number, how do you extract the number from the string? The answer depends on what type of number it is.

Consider the following code fragment. It begins by extracting the substring "3.14" from the string a, and it then converts this substring to the corresponding number, 3.14. Finally, the program reports the result of multiplying this number by 2.0.

    String a = "Pi is 3.14 approx.";
    String b = a.substring( 6, 10 );
    double c = Double.parseDouble( b );

    double d = c * 2.0;

    System.out.println( "The value of d is " + d );

The statement that causes the string-to-number conversion to take place involves the method Double.parseDouble, which takes a single argument. If that argument is a string that names a floating point value, then the method returns the corresponding floating point value.

Computer scientists refer to the process of interpreting a string in order to yield one or more (usually non-string) values as parsing. That is why the name of this method � and that of the corresponding method for integers (see below) � includes the word "parse".

To convert a double to a String, we can deliberately provoke the automatic conversion that occurs when we concatenate a string and a number by concatenating the floating point number with the empty string. The expression "" + 5.6, for example, evaluates to the string "5.6". Alternatively, we can use the method Double.toString, which is specifically provided for this purpose. This method takes one argument, namely, the floating point value that is to be converted. Then, for example, Double.toString( 5.6 ) evaluates to the string "5.6".

Similar methods are available for integers. In particular, the method Integer.parseInt takes one argument, which must be a string that names an integer, and converts it to an int. So, for example, (Integer.parseInt( "5" ) + 2) evaluates to 7.

Conversely, to convert an int to a string, either concatenate the integer with the empty string or use the one-argument Integer.toString method.
